CTATComboBox Example

A select example.

Select "yes": 
<div id="combo1" class="CTATComboBox">
  <option>----</option>
  <option>yes</option>
  <option>no</option>
</div>
A select where the options are set with an attribute: 
<div id="combo2" class="CTATComboBox"
     data-ctat-labels="----,oui,non">
</div>
A select using both ways of setting options: 
<div id="combo2" class="CTATComboBox"
     data-ctat-labels="oui;non"
     data-ctat-split-on=";">
  <option>----</option>
</div>
